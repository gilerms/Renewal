// Hocus-Pocus (Abracadabra) Banco de Dados de Habilidades Castáveis.
//
//Estrutura do banco de dados:
//SkillID, DummyName, ProbabilityPerLvl
//
//01. SkillID Skill ID a ser lançado pelo hocus pocus.
//02. DummyName Nome da habilidade (informativa, não usada pelo servidor).
//03. ProbabilityPerLvl Não é uma taxa! Chance em que a habilidade é selecionada em comparação
//com outras entradas prováveis.
//
// NOTA:
//- A habilidade é escolhida aleatoriamente de todo o banco de dados e testada para taxa. Se isso
//não tem sucesso nessa taxa, outra habilidade é escolhida e testada. Isso continua
//até que uma habilidade seja bem sucedida. Habilidades específicas para abracadabra têm uma chance diferente de ocorrer
//dependendo do nível de habilidade usado. Todas as outras habilidades têm uma chance igual e aparecem.
//nível 1 em diante.
//- Para remover a entrada importando, coloque "clear" (sem aspas) em DummyName
